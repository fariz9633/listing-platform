================================================================================
                    HOW TO ACCESS ADMIN, PROVIDER & GUEST PAGES
================================================================================

STEP 1: START THE SERVER
-------------------------
cd listing-platform
php artisan serve

Server will start at: http://localhost:8000


STEP 2: OPEN THE LOGIN PAGE
----------------------------
Go to: http://localhost:8000/login

You'll see buttons to login as:
- Admin (red button)
- Provider (green buttons)
- Guest (blue buttons)

ALL PAGES BY ROLE
================================================================================

PUBLIC PAGES (No Login Required):
----------------------------------
Home/Listings:          http://localhost:8000
Browse Listings:        http://localhost:8000/listings
View Listing:           http://localhost:8000/listings/{slug}
Send Enquiry:           http://localhost:8000/listings/{slug}/enquire


ADMIN PAGES (Login as admin@example.com first):
------------------------------------------------
Moderation Queue:       http://localhost:8000/admin/moderation
Review Listing:         http://localhost:8000/admin/moderation/{id}

What Admin Can Do:
‚úì View all pending listings
‚úì Approve listings
‚úì Reject listings (with reason)
‚úì Suspend listings
‚úì View moderation history


PROVIDER PAGES (Login as sarah@example.com first):
---------------------------------------------------
Provider Dashboard:     http://localhost:8000/provider/dashboard
Create Listing:         http://localhost:8000/provider/listings/create
Edit Listing:           http://localhost:8000/provider/listings/{id}/edit

What Provider Can Do:
‚úì View all their listings
‚úì Create new listings (saved as draft)
‚úì Edit draft listings
‚úì Submit listings for moderation
‚úì See rejection reasons
‚úì Delete draft listings


GUEST PAGES (Login as guest1@example.com or browse without login):
-------------------------------------------------------------------
Browse Listings:        http://localhost:8000/listings
View Listing:           http://localhost:8000/listings/{slug}
Send Enquiry:           http://localhost:8000/listings/{slug}/enquire

What Guest Can Do:
‚úì Browse all approved listings
‚úì Search and filter listings
‚úì View listing details
‚úì Send enquiries to providers

TROUBLESHOOTING
================================================================================

Problem: "User not found" error
Solution: Make sure you ran: php artisan migrate:fresh --seed

Problem: Can't access admin/provider pages
Solution: Login first using the /login page or test-login URLs

Problem: No listings showing
Solution: Run: php artisan migrate:fresh --seed

Problem: Server not running
Solution: Run: php artisan serve
